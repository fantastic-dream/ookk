# ookk

> simple like psi repo, but focus in what i need. and use my proxy request.

## Use

```

  Usage
    $ ookk <url>

  Options
    --key        Google API Key.
    --strategy   Strategy to use when analyzing the page: mobile|desktop

  Example
    $ ookk https://zwkang.com --strategy=mobile

```
